## Notes

- Logo, description, and home page are grabbed from Twitter profile.
- Number of reviews and average ratings are scraped from
    - Facebook
    - Switchup
    - Yelp
    - Techendo
    - Course Report
    - Google+
    - Quora
- Social score is calculated from
    - Facebook
    - Google+
    - Quora
    - Twitter
